# Sales & Revenue Analysis Dashboard

## Overview

This project is a Sales & Revenue Analysis Dashboard developed using Python and Power BI. The objective is to analyze sales performance, identify revenue trends, evaluate product performance, and generate actionable business insights through interactive visualizations.

## Objectives

* Clean and validate the dataset using Python.
* Perform Exploratory Data Analysis (EDA).
* Build an interactive Power BI dashboard.
* Analyze sales, profit, regional performance, and product performance.
* Generate business insights to support decision-making.

## Dataset

The project uses the Superstore Sales Dataset containing information about:

* Orders
* Customers
* Products
* Sales
* Profit
* Discounts
* Regions
* Categories
* Shipping Details

## Tools & Technologies

* Python
* Pandas
* Jupyter Notebook
* Power BI
* DAX

## Data Cleaning

The following preprocessing steps were performed:

* Checked dataset dimensions
* Verified data types
* Checked missing values
* Removed duplicate records
* Converted date columns
* Created Year, Month, and Month Number columns
* Validated numerical data using descriptive statistics

## Dashboard KPIs

## Dashboard Visualizations

* Monthly Sales Trend
* Sales by Category

## Interactive Filters

* Year
* Ship Mode
* Category
* Region

## Key Business Insights

* Technology generated the highest sales and profit among all product categories.
* Furniture generated high sales but comparatively low profit.
* The West region recorded the highest sales and profitability.
* Sales showed strong growth after 2015, with 2017 achieving the highest annual revenue.
* November and December recorded the highest monthly sales.
* Higher discounts generally reduced profitability.
* A small number of products contributed significantly to total revenue.
